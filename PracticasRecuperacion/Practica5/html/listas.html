Listas convencionales

Las listas se crean con las etiquetas <ul> </ul> (listas desordenadas) y <ol> </ol> (listas ordenadas). Entre las etiquetas de apertura y cierre, que es obligatoria, cada elemento de la lista se tiene que crear utilizando las etiquetas <li> </li>. La etiqueta de cierre </li> es opcional, aunque recomendable. <li> es un objeto de lista que puede contener otros elementos de bloque como párrafos, imágenes, enlaces o directamente el texto. A continuación puedes ver un ejemplo: 

    <ul>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ul>

El resultado sería:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Atributos de <ul>:

  • type: Indica el tipo de símbolo utilizado y puede tomar los valores "disc" (disco), "square" (cuadrado) y "circle" (círculo). Está en Desuso
  • compact: Si aparece indica al navegador que la ista debe compactarse.

Atributos de <ol>:

  • type: Indica el tipo de secuencia de lista y puede tomar los valores "1" (números enteros), "a" (letras minúsculas), "A" (letras mayúsculas), "i" (números romanos minúsculas), "I" (números romanos mayúsculas). Está en Desuso
  • start: Indica el número del primer elemento de la lista por el que debe empezar. Está en desuso.

Se puede personalizar las opciones de las listas. Para ello, consulta la guía de referencia rápida de HTML de los apuntes.

Listas anidadas

Para realizar listas anidadas hay que incluir una estructura <ul> o <ol> dentro del elemento <li> que contendrá la lista anidada. Por ejemplo:

    <ul>
        <li>Elemento 1</li>
        <ul>
            <li>Elemento 1.1</li>
            <li>Elemento 1.2</li>
            <li>Elemento 1.3</li>
        </ul>
        <li>Elemento 2</li>
    </ul>

El resultado sería:

  • Elemento 1
    • Elemento 1.1
    • Elemento 1.2
    • Elemento 1.3
  • Elemento 2

Listas de definición

Las listas de definiciónse crean con las etiquetas <dl> </dl>. La diferencia de este tipo de listas con las anteriores es que tienen dos partes: el término de la definición, que se crea utilizando las etiquetas <dt> </dt>, y la definición, que se crea utilizando las etiquetas <dd> </dd>. Todo el texto aparece con sangría. La etiqueta de cierre </dl> es obligatoria. Las etiquetas de cierre </dt> y </dd> son opcionales. A continuación puedes ver un ejemplo: 

    <dl>
        <dt>Coche</dt>
        <dd>Vehículo automóvil destinado al transporte de personas</dd>

        <dt>Tren</dt>
        <dd>Medio de transporte que circula sobre railes</dd>
    </dl>

El resultado sería:

Coche
Vehículo automóvil destinado al transporte de personas
Tren
Medio de transporte que circula sobre railes
