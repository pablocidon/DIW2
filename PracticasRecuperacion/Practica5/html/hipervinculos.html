Hipervínculos

Los hipervínculos son enlaces que permiten acceder a otro documento cuando se pincha sobre ellos. Son los que posibilitan navegar de una página web a otra. Para insertar un hipervínculo en una página web se utiliza la etiqueta <a>. Esta etiqueta admite varios parámetros que se pueden consultar en la guía de referencia rápida.

La etiqueta de cierre es obligatoria.

Es un elemento de línea por lo que puede estar dentro de un párrafo, lista, etc..., a su vez <a> puede contener otros elementos en línea excepto a sí mismo.

Atributos:

  • name: Sirve para nombrar un ancla o marcador que será el destino del enlace.
  • href: Sirve para especificar la URI (Uniform Resource Identifier) destino del hipervínculo.
  • target: Establece el marco (frame) de destino (por defecto se toma la ventana actual). Valores posible: _blank, _self, _parent, _top.

Su estructura básica es:

    <a href="destino"Texto del enlace</a>

El valor de destino puede ser:

  • Una página web: http://www.google.es
  • Una página web almacenada localmente (ruta global): c:\datos\pagina1.html
  • Una página web almacenada localmente (ruta relativa): ./carpeta1/pagina1.html
  • Un documento (pdf, imagen, archivo ejecutable, zip,...) almacenado localmente (ruta global): c:\datos\documento1.pdf
  • Un documento (pdf, imagen, archivo ejecutable, zip,...) almacenado localmente (ruta relativa): ./documentos/documento1.pdf

¡Ojo! En las rutas globales, el separador entre carpetas es distinto para sistemas operativos Windows (se utiliza \) y Linux (se utiliza /). Además, no es conveniente trabajar con nombres de carpetas que contengan caracteres no ingleses (eñes, acentos,...) o espacios en blanco solo suelen dar disgustos.

En cuanto a las rutas relativas, hay que recordar que el directorio punto (.) significa directorio actual y el directorio dos puntos (..) la carpeta superior. Por ejemplo, la ruta ./carpeta1/doc1.html hace referencia al archivo doc1.html que se encuentra dentro de una carpeta llamada carpeta1 que está en el mismo directorio que la página actual.

Anclas

Es posible crear enlaces que nos lleven a un punto concreto dentro de un determinado documento. Para ello se utilizan las anclas.

En primer lugar es necesario definir dentro del documento HTML el punto al que queremos llegar. Para ello se utiliza también la etiqueta <a>, pero con el atributo name, que contendrá el nombre que le queramos dar al ancla: 

    <a href="nombreAncla">Texto del enlace</a>

Una vez establecida el ancla, para llegar a ella crearemos un enlace que contendrá como destino el nombre del ancla a la que queramos llegar precedida del carácter #. Si el ancla a la que queremos ir está en otra página distinta, el destino del enlace será el nombre de la página seguido a continuación del símbolo # y el nombre del ancla. Por ejemplo: 

  <a href="#ancla1">Enlace al ancla dentro de la misma página</a>
  <a href="./pagina2.html#ancla1">Enlace al ancla que está dentro
  de la página 2</a>

También es posible utilizar como marcador de posición un elemento ya existente en la página, en lugar de crear un nuevo marcador.
